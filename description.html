Description of the site

This is a personal portfolio collection site which is made using HTML5 and CSS3.This site consists of five pages.These page can be easily navigated by clicking on the option displayed on the navigation bar on the left side of each page.

The first page is the homepage or the landing page where there is a short introduction to the page and about myself.

The second page is the 'About Me' page where I have included description about myself.

The third page is the 'Gallery' page with images and a short description of each images.

The fourth page is the 'Contact Me' page where a form is available with few details to fill up. This page can also be opened by clicking on the button 'Hire Me' which is placed on the homepage.
